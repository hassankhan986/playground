Try these commands on console

$('li') This will find all the "li" elements
$('li').first() This will find the first "li" element
$('li').last() This will find the last "li" element
$('li').first().hide() This will hide the first "li" element
$('li').first().show() This will show the first "li" element
$('li').eq(0) Returns first index "li" element
$('li').eq(1) Returns second index "li" element
$('ul:first') Returns first "ul" element
$('ul:first').children() Selects first "ul" and then get all its children elements
$('li:first').siblings() Selects all the siblings of first 'li'
$('li:first').parent() Select the parent of first 'li'
$('li').eq(1).prev() Select the second 'li' and then select previous 'li' which is the first 'li'
$('li').eq(1).next() Select the second 'li' and then select next 'li' which is the third 'li'
