flex-shrink works opposite of flex-grow. Let's say we have a situation where each items takes equal space in a parent flex. Then flex shrink could shrink an item by the ratio number given. 
.flex-container {
    display: flex;
}

.flex-item {
    width: 30%;
    height: 30px;
}

.flex-item:nth-child(1) {
    flex-shrink: 3;
}

.flex-item:nth-child(3) {
    flex-shrink: 4;
}
