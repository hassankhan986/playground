align-self works just like align-item or align-content. The difference is that it aligns itself according to the items group its in. 
.flex-container {
    height: 300px;
    display: flex;
    flex-wrap: wrap;
}

.flex-item {
    width: 30%;
    height: auto;
}

.flex-item:nth-child(3) {
    align-self: center;
}
