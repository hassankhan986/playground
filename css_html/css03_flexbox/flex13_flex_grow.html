• flex-grow works on the same concept like android layout weight works.
  • By default flex-grow value is 0. 0 means don't grow.
  • flex-grow value should be greater than zero to take effect.
  • The more flex-grow value is compare to other item's value the more it will take space.
.flex-container {
    display: flex;
}

.flex-item {
    flex-grow: 1;
    height: 30px;
    line-height: 30px;
}

.flex-item:nth-child(2) {
    flex-grow: 2;
}

.flex-item:nth-child(5) {
    flex-grow: 4;
}
