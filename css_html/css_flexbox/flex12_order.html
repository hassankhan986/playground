Every flex item has order. If not specified then its 0. If multiple item have same order number then item's order depends on where they appear in the code. 
.flex-container {
    display: flex;
}

.flex-item {
    width: 30%;
    height: 30%;
    line-height: 50px;
}

.flex-item:nth-child(5) {
    order: -8;
}

.flex-item:nth-child(2) {
    order: 3;
}

.flex-item:nth-child(4) {
    order: 2;
}
